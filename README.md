# EduTrack Pro — Student Management System

## 🚀 Quick Setup (3 Steps)

Check here to visit the Project: https://student-management-details.vercel.app/

### Step 1 — Download & Extract
Download all 3 files and place them in ONE folder:
```
StudentMS/
  ├── index.html
  ├── style.css
  └── app.js
```

### Step 2 — Open in VS Code
```
File → Open Folder → Select the StudentMS folder
```
Install the **Live Server** extension (by Ritwick Dey) from Extensions panel.

### Step 3 — Run
Right-click `index.html` → **"Open with Live Server"**  
OR press `Ctrl+Shift+P` → type `Live Server: Open with Live Server`

Your browser opens at `http://127.0.0.1:5500/index.html` ✅

## 📋 Features

| Feature | Description |
|---|---|
| ➕ Add Student | Full form with 3 tabs: Basic, Academic, Contact |
| 🆔 Auto Roll ID | Unique ID like `BT24CSE0001` generated on save |
| ✏️ Edit Student | Update any field at any time |
| 👁️ View Profile | Rich profile card with all details |
| 🗑️ Delete | Confirm-before-delete protection |
| 📅 Attendance | Mark Present/Absent per date, per student |
| 📊 Marks | Enter Internal + External → auto-calculates Total, Grade, GP |
| 📚 Subjects | Add/edit subjects per branch per semester |
| 📈 Reports | Branch stats, grade distribution, attendance summary |
| 🔔 Notifications | Activity log with badge counter |
| 🔍 Search | Live global search across all students |
| ⊞ Grid / List View | Toggle between card and table view |
| 💾 Persistent | All data saved in browser localStorage |

## 🎯 Roll ID Format
`[Program][Year][Branch][Sequence]`
- Example: `BT24CSE0001` = B.Tech, 2024, CSE, first student
- Example: `MT24AIR0001` = M.Tech, 2024, AI, first student
